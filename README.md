Overview :

This project focuses on performing Exploratory Data Analysis (EDA) to understand, clean, and analyze real-world datasets. 
The main goal is to identify patterns, trends, and insights using statistical methods and visualizations.

Objectives :

Clean and preprocess raw data
Handle missing and inconsistent values
Perform statistical analysis
Visualize data using graphs and charts
Extract meaningful business insights

Tools & Technologies :
Python 
Pandas & NumPy – Data manipulation
Matplotlib & Seaborn – Data visualization
Jupyter Notebook – Analysis environment
